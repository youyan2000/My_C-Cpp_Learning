# 双向链表

## 一、实验目的：

理解线性结构的基本概念，掌握两种基本的存储结构：顺序存储结构（顺序表）和链式存储结构（单链表、双向链表）。用 C 语言实现两种结构下的基本操作（创建、添加、删除、遍历输出和原地逆转），巩固强化用 C 语言进行模块化程序设计的基本方法和能力，并复习加强 C 语言中结构体和指针的使用。要求界面友好，功能正确，算法高效。

## 二、实验内容：

1. 用模块化设计思想，所有功能均需用函数实现双向链表的不同操作。
2. CreateList 从键盘输入 10 个数，将其存入表中，以 -1 表示结束（-1 不存入表中）。
3. AddtoList 在表中指定位置添加一个元素，输出添加成功后的线性表数据。
4. DelListData 删除表中所有的负数，输出操作成功后的线性表数据。
5. InvertList 逆转线性表中的元素，输出操作成功后的线性表数据。

## 三、验机标准：

1. 实现 CreateList、AddtoList、DelListData、ReviseList、OutputList 五个函数。
2. 界面友好，程序健壮（测试验收时面对非期望数据无死机现象，且有提示）。
3. 用双向链表实现，功能完整，达到要求。
