# 数据类型

## 数据类型小结
| 类型大类       | 子类/类型名称                             | 类型说明                             | 常见大小（典型）         | 备注                                          |
| -------------- | ----------------------------------------- | ------------------------------------ | ------------------------ | --------------------------------------------- |
| 基本内置类型   | `char`(character)                         | 字符型，用于存储字符或小整数         | 1字节                    | 可带 `signed`/`unsigned` 修饰                 |
|                | `short` (short integer)                   | 短整型                               | 2字节                    | 可带符号修饰                                  |
|                | `int`(integer)                            | 整型，最常用的整数类型               | 4字节                    | 默认为 `signed`                               |
|                | `long` (long integer)                     | 长整型                               | 4或8字节（平台相关）     | 可带符号修饰                                  |
|                | `long long`(long long integer)            | 长长整型                             | 8字节                    | C++11 起支持，用于大整数                      |
|                | `float`(floating-point)                   | 单精度浮点数                         | 4字节                    | 精度约6-7位有效数字                           |
|                | `double`(double-precision floating-point) | 双精度浮点数                         | 8字节                    | 精度约15-16位有效数字                         |
|                | `long double`                             | 长双精度浮点数                       | 平台相关（通常 ≥8 字节）| 更高精度，平台依赖                            |
|                | `bool`(boolean)                           | 布尔类型                             | 通常1字节                | 取值为 `true` 或 `false`                      |
|                | `wchar_t`                                 | 宽字符类型                           | 平台相关（通常2或4字节） | 用于大字符集                                  |
|                | `char16_t`                                | UTF-16 字符类型                      | 固定2字节                | C++11 引入                                    |
|                | `char32_t`                                | UTF-32 字符类型                      | 固定4字节                | C++11 引入                                    |
|                | `void`                                    | 空类型                               | 无                       | 不表示任何值，用于函数返回类型或 `void*` 指针 |
| 派生类型       | 数组 (Array)                              | 存储相同类型元素的连续内存块         | 取决于元素个数和类型     | 格式：`type name[size]`                       |
|                | 指针 (Pointer)                            | 存储变量的内存地址                   | 通常4或8字节（平台相关） | 类型为 `type*`                                |
|                | 引用 (Reference)                          | 变量的别名                           | 无独立大小               | 初始化后不可更改绑定对象                      |
|                | 函数类型 (Function)                       | 表示函数的类型                       | —                       | 可用于函数指针或回调                          |
| 用户自定义类型 | 结构体 (struct)                           | 将不同类型的数据组合成一个整体       | 各成员大小之和（含对齐） | C/C++ 中默认公有成员                          |
|                | 类 (class)                                | 面向对象的核心，支持封装、继承、多态 | 同结构体                 | C++ 中默认私有成员                            |
|                | 联合体 (union)                            | 多个成员共享同一块内存               | 最大成员的大小           | 同一时间只能使用一个成员                      |
|                | 枚举 (enum)                               | 定义一组命名的整数常量               | 通常为 `int` 大小        | C++11 支持强类型 `enum class`                 |

不同数据类型的唯一本质区别，在于占有内存的大小
