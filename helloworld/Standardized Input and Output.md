# 标准化输入输出
## printf
- 作用：将指定数据按照自定义格式，输出到屏幕（标准输出设备）。
- 基本语法：`printf("%[flags][width][.prec][hIL]type", output);`
- 说明：`[]` 中的内容为可选参数，核心是 `%` 开头的格式控制符，用于指定输出数据的类型和格式；`output` 为要输出的变量、常量或表达式（可多个，用逗号分隔）。

### 1. flags（格式标志，可选）
用于控制输出的对齐方式、符号显示等
- `-`：左对齐（默认右对齐），示例：`printf("%-5d", 123);` 输出 `123  `（占5位，左对齐）
- `+`：强制显示正负号（正数显+，负数显-），示例：`printf("%+d", 123);` 输出 `+123`
- `0`：右对齐时，用0填充空白位（而非空格），示例：`printf("%05d", 123);` 输出`00123`
- `#`：对八进制（%o）、十六进制（%x/%X）格式，强制显示前缀（0、0x/0X），示例：`printf("%#x", 10);` 输出 `0xa`
- `space`：正数前显示空格（代替+号），负数正常显-，示例：`printf("% d", 123);` 输出 ` 123`

### 2. width（输出宽度，可选）
- `number`：指定输出内容的总宽度（包含数字、符号、小数点等），不足补空格（或0，配合0标志），超出则按实际长度输出
- `*`：占位输入

e.g.
- `printf("%5d", 123);` 输出 `  123`（占5位，右对齐，空白补空格）
- `printf("%5s", "abc");` 输出 `  abc`（字符串同样适用）

### 3. .prec（精度，可选）
格式：`.数字`，作用根据type（数据类型）不同而变化，核心是控制“有效位数”或“小数位数”。
- 对整数（d/o/x/u）：指定最少输出的有效数字位数，不足补0，超出按实际输出；若为0，且输出值为0，则不输出任何数字。
- 对浮点数（f/e/E/g/G）：指定小数点后保留的位数，多余部分四舍五入，不足补0。
- 对字符串（s）：指定输出的最大字符数，超出部分截断。

e.g.
- `printf("%.3d", 12);` 输出 `012`（最少3位有效数字，补0）
- `printf("%.2f", 3.1415);` 输出`3.14`（保留2位小数，四舍五入）
- `printf("%.3s", "hello");` 输出 `hel`（最多输出3个字符）

### 4. hIL（长度修饰符，可选）
用于指定输出数据的长度，匹配不同类型的变量
- `hh`：char(1)
- `h`：short(2)，配合 type 为 d、o、x、u，示例：`printf("%hd", (short)123);`
- `l`：long/double(4)，配合 type 为 d、o、x、u、f，示例：`printf("%ld", (long)123456);`
- `ll`：long long(8)，配合 type 为 d、o、x、u，示例：`printf("%lld", (long long)123456789);`
- `L`：long double(16)

### 5. type（格式类型，必选）
| type符号 | 对应数据类型                        | 说明                                           | 示例                                         |
| -------- | ----------------------------------- | ---------------------------------------------- | -------------------------------------------- |
| %d       | 十进制整型（int）                   | 最常用，输出整数                               | printf("%d", 123); → 123                     |
| %o       | 八进制整型                          | 无符号，不显示前缀0（加#可显示）               | printf("%#o", 10); → 012                     |
| %x/%X    | 十六进制整型                        | 无符号，%x小写a-f，%X大写A-F（加#可显示0x/0X） | printf("%x", 10); → a；printf("%X",10);→A    |
| %u       | 无符号十进制整型                    | 只输出非负数，负数会按无符号数解析             | printf("%u", -1); → 4294967295（32位系统）   |
| %f       | 单精度/双精度浮点型（float/double） | 默认保留6位小数，配合.prec控制小数位数         | printf("%.2f", 3.14); → 3.14                 |
| %e/%E    | 科学计数法表示浮点型                | %e小写e，%E大写E，默认保留6位小数              | printf("%e", 123.45); → 1.234500e+02         |
| %g/%G    | 浮点型（自动选择f或e格式）          | 去掉多余的0，更简洁                            | printf("%g", 123.4500); → 123.45             |
| %c       | 字符型（char）                      | 输出单个字符                                   | printf("%c", 'A'); → A                       |
| %s       | 字符串（char[]/char*）              | 输出到'\0'结束符为止                           | printf("%s", "hello"); → hello               |
| %p       | 指针类型                            | 输出指针的十六进制地址                         | printf("%p", &a); → 0x7ffdxxxx（随系统变化） |

## scanf
- 作用：从键盘（标准输入设备）读取数据，按照指定格式存入对应变量中。
- 基本语法：`scanf("%[flags]type", &input);`
- 说明：`&` 是取地址符，除了字符串（char[]），其他变量必须加 &（获取变量的内存地址，才能将输入值存入）；可同时读取多个变量，用逗号分隔格式符和变量。

### 1. flags（格式标志，可选，较少用）

- `*`：跳过当前输入项，不存入任何变量，示例：`scanf("%*d%d", &a);` 输入 `123 456`，只读取456存入a，跳过123
- `number`：指定输入内容的总宽度（包含数字、符号、小数点等），不足补空格（或0，配合0标志），超出则按实际长度输出
- `hh`：char(1)
- `h`：short(2)，配合 type 为 d、o、x、u
- `l`：long/double(4)，配合 type 为 d、o、x、u、f
- `ll`：long long(8)，配合 type 为 d、o、x、u
- `L`：long double(16)

### 2. type（格式类型，必选）
与printf的type基本一致，对应要读取的数据类型，但scanf的type用于“匹配输入数据”，必须和变量类型完全对应。
| type符号 | 对应变量类型  | 说明                                                                        | 示例                                                    |
| -------- | ------------- | --------------------------------------------------------------------------- | ------------------------------------------------------- |
| %d       | int*          | 读取十进制整数                                                              | int a; scanf("%d", &a); 输入123 → a=123                 |
| %o/%x/%X | int*          | 读取八进制、十六进制整数，自动转换为十进制存入变量                          | int a; scanf("%x", &a); 输入a → a=10                    |
| %u       | unsigned int* | 读取无符号十进制整数                                                        | unsigned int a; scanf("%u", &a);                        |
| %f       | float*        | 读取浮点型数据（输入时可写小数或科学计数法）                                | float a; scanf("%f", &a); 输入3.14 → a=3.14             |
| %lf      | double*       | 读取双精度浮点型数据（注意：printf用%f，scanf用%lf）                        | double a; scanf("%lf", &a); 输入3.1415 → a=3.1415       |
| %c       | char*         | 读取单个字符（会读取空格、回车等空白字符）                                  | char c; scanf("%c", &c); 输入A → c='A'                  |
| %s       | char[]/char*  | 读取字符串，以空格、回车、制表符为分隔符，自动在末尾加'\0'结束符（无需加&） | char str[10]; scanf("%s", str); 输入hello → str="hello" |

## 逃逸字符
作用：表示一些“无法直接输入”的字符（如回车、制表符），以 `\` 开头，后面跟特定字符，在printf、字符串中常用。
| 逃逸字符               | 含义                                                    | 示例（printf输出效果）                                |
| ---------------------- | ------------------------------------------------------- | ----------------------------------------------------- |
| \n                     | 换行符（ newline ）                                     | printf("hello\nworld"); → 换行显示hello和world        |
| \t                     | 制表符（ tab ），相当于按一次Tab键（对齐用）            | printf("name\tage"); → name    age（中间空4-8个空格） |
| \\|输出反斜杠本身（\） | printf("\\"); → 屏幕显示 \                              |
| \''                    | 输出单引号（'）                                         | printf("\'A\'"); → 屏幕显示 'A'                       |
| \"                     | 输出双引号（"）                                         | printf("\"hello\""); → 屏幕显示 "hello"               |
| \r                     | 回车符（ carriage return ），光标回到当前行开头         | printf("hello\rworld"); → 输出world，覆盖hello        |
| \b                     | 退格符（ backspace ），光标向前退一格（删除前一个字符） | printf("hell\bo"); → 输出 helix（删除l，替换为o）     |

## 其他神秘特性
1. printf的格式控制符和output的数量、类型必须一一对应，否则输出乱码；scanf同理，格式符和变量类型、数量必须匹配，否则读取失败。
2. 浮点数输入：float用%f，double用%lf；浮点数输出：float和double都可用%f（部分编译器支持%lf，但推荐统一用%f）。
3. 逃逸字符只能在字符串、printf中生效，单独写无意义；比如 '\n' 是换行，而 n 只是普通字符。
4. getchar/putchar是字符专用输入输出函数，无任何格式化功能，只能读取 / 输出 1 个字符
